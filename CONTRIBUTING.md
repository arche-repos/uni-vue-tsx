# 贡献指南

感谢您对 uni-vue-tsx 模板项目的关注！

## 如何贡献

### 报告问题
- 使用 GitHub Issues 创建问题报告
- 请提供复现步骤和预期行为
- 包含相关环境信息

### 提交改进建议
1. Fork 本仓库
2. 创建功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 创建 Pull Request

### 代码规范
- 遵循现有的代码风格
- 使用 ESLint 和 Prettier 自动格式化
- 确保所有测试通过

## 开发流程

1. 安装依赖：`yarn install`
2. 开发：使用 `yarn dev:h5` 或 `yarn dev:mp-weixin`
3. 测试：运行 `yarn lint` 和 `yarn type-check`
4. 构建：确保 `yarn build:h5` 和 `yarn build:mp-weixin` 都能成功
